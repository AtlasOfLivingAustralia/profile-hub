Creating Profiles

Only Collection Administrators and Profile Editors can create new profiles.

  1. From the Collection home page, select the 'Add a new profile' button.
  2. Enter the name you wish to give your profile. Names must be unique within a collection.
  3. Click 'Check Name'
  4. The system will attempt to match the profile name you provided with a known scientific name. See the Profile Name section for more details.
  5. Follow the prompts to match the name
  6. Click Create Profile
  7. You will now be taken to the Edit Profile page, where you can enter the rest of the information for your profile.

NOTE: Your new profile is visible to general users as soon as you click Create Profile.

Copying an existing profile

You can copy the content of another profile in your collection when creating a new profile. Only the user-defined attributes are copied: name matching, images, and data that is automatically retrieved from other ALA systems will NOT be copied.

  1. From the Collection home page, click the drop-down arrow beside the 'Add a new profile' button, then select 'Copy an existing profile'
  2. Enter the name of the profile you wish to copy. This can be any profile within your collection. The system will display a list of matching profile names as you type.
  3. Select the profile you wish to copy from the list
  4. Enter the name you wish to give your profile. Names must be unique within a collection.
  5. Click 'Check Name'
  6. The system will attempt to match the profile name you provided with a known scientific name. See the Profile Name section for more details.
  7. Follow the prompts to match the name
  8. Click Create Profile
  9. You will now be taken to the Edit Profile page, prefilled with the user-defined attributes from the profile you copied.

NOTE: Your new profile is visible to general users as soon as you click Create Profile.
